# badusb

## Setup
- Hold BOOTSEL button while plugging Pico into computer
- Drag pico-badusb.uf2 into Pico storage

## Reset
- Hold BOOTSEL button while plugging Pico into computer
- Drag flash_nuke.uf2 into Pico storage
